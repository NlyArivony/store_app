- build the docker image :
    -> docker build -t rest-apis-flask-python .
- run the docker image :
    -> docker run -p  5005:5000 rest-apis-flask-python
- to solve the problem while updating the code and rerun docker build (use a volume):
    -> docker run -dp  5005:5000 -w /app -v "$(pwd):/app" rest-apis-flask-python


/////////////////////////////////////////////////////////////////////////////////////////
-initialise db with flask migrate to create a migration folder:
    -> flask db init

-create the first migration:
    -> flask db migrate

-apply migration:
    -> flask db upgrade

-rollback to previous migration:
    -> flask db downgrade

        -> docker build -t rest-apis-flask-python .
    - run the rq worker :
        -> docker run -w /app rest-apis-flask-python sh -c "rq worker -u REDIS_URL emails"
